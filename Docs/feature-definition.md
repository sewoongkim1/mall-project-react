# 서비스 기능 정의서

> 버전: v1.0 | 작성일: 2026년 4월

---

## 1. 사용자 역할

| 역할 | 설명 | 주요 권한 |
|------|------|----------|
| 구매자 (Buyer) | 의류를 검색하고 구매하는 일반 사용자 | 상품 탐색, 주문, AI 추천, 리뷰 |
| 셀러 (Seller) | 상품을 등록하고 판매하는 브랜드/공급사 | 상품 등록/관리, 주문 처리, 매출 조회 |
| 관리자 (Admin) | 플랫폼 전체를 운영/관리하는 내부 팀 | 사용자 관리, 정산, 통계, 시스템 설정 |

---

## 2. 인증/회원 기능

| 기능 ID | 기능명 | 설명 | 우선순위 |
|---------|--------|------|---------|
| AUTH-001 | 이메일 회원가입 | 이메일, 비밀번호, 닉네임으로 계정 생성 | 필수 |
| AUTH-002 | 이메일 로그인 | 이메일 + 비밀번호 인증, JWT 발급 | 필수 |
| AUTH-003 | 소셜 로그인 | 카카오, 구글 OAuth2 연동 | 권장 |
| AUTH-004 | 비밀번호 재설정 | 이메일 인증 링크를 통한 비밀번호 변경 | 필수 |
| AUTH-005 | 회원 권한 분리 | 구매자/셀러/관리자 Role 기반 접근 제어 | 필수 |
| AUTH-006 | 취향 정보 입력 | 회원가입 후 선호 스타일/사이즈/브랜드 설문 | 필수(AI) |

---

## 3. 상품 관리 기능 (셀러)

| 기능 ID | 기능명 | 설명 | 우선순위 |
|---------|--------|------|---------|
| PRD-001 | 상품 등록 | 상품명, 설명, 카테고리, 가격, 이미지 등록 | 필수 |
| PRD-002 | 상품 수정/삭제 | 등록된 상품 정보 업데이트 및 삭제 | 필수 |
| PRD-003 | 이미지 업로드 | 다중 이미지 업로드 (최대 10장) | 필수 |
| PRD-004 | 사이즈/색상 재고 | SKU 단위 재고 수량 관리 | 필수 |
| PRD-005 | 카테고리/태그 | 상품 분류 및 AI 추천용 스타일 태그 설정 | 필수(AI) |
| PRD-006 | 상품 상태 관리 | 판매중/품절/숨김/삭제 상태 전환 | 필수 |
| PRD-007 | 셀러 스토어 페이지 | 셀러 브랜드 소개, 상품 목록 노출 | 권장 |

## 4. 상품 탐색 기능 (구매자)

| 기능 ID | 기능명 | 설명 | 우선순위 |
|---------|--------|------|---------|
| SRCH-001 | 상품 목록 페이지 | 카테고리별 상품 그리드, 무한 스크롤 | 필수 |
| SRCH-002 | 상품 상세 페이지 | 이미지 갤러리, 상세 설명, 리뷰, 관련 상품 | 필수 |
| SRCH-003 | 키워드 검색 | 상품명, 브랜드, 태그 기반 전문 검색 | 필수 |
| SRCH-004 | 필터 및 정렬 | 가격/카테고리/사이즈/색상/브랜드 필터링 | 필수 |
| SRCH-005 | 정렬 옵션 | 최신순, 인기순, 가격순, 리뷰순 | 필수 |
| SRCH-006 | 찜하기 | 관심 상품 저장, AI 데이터로 활용 | 필수(AI) |

## 5. 장바구니/결제

| 기능 ID | 기능명 | 설명 | 우선순위 |
|---------|--------|------|---------|

| CART-005 | 주문서 작성 | 배송지 입력, 요청사항, 최종 금액 확인 | 필수 |

## 6. 주문/배송/리뷰

|---------|--------|------|---------|

| ORD-002 | 주문 상태 관리 | 주문접수→결제완료→배송준비→배송중→완료 | 필수 |
| ORD-003 | 셀러 주문 대시보드 | 셀러별 주문 현황 관리 | 필수 |

| DEL-001 | 운송장 등록 | 셀러가 택배사, 운송장 번호 입력 | 필수 |
| DEL-002 | 배송 현황 조회 | 구매자 마이페이지에서 실시간 추적 | 필수 |

| LOG-001 | 행동 로그 수집 | 클릭/찜/검색/구매 이력 DB 저장 (AI용) | 필수(AI) |

## 7. AI 개인화 추천

| 기능 ID | 기능명 | 설명 | 우선순위 |
|---------|--------|------|---------|

## 8. 관리자(Admin) 기능

| 기능 ID | 기능명 | 설명 | 우선순위 |
|---------|--------|------|---------|
| ADM-001 | 회원 관리 | 사용자 목록 조회, 정지, 탈퇴 처리 | 필수 |

| ADM-005 | 정산 관리 | 셀러별 정산 내역, 수수료 관리 | 필수 |
| ADM-006 | AI 추천 성능 | 추천 CTR, CVR 리포트 | 권장 |
