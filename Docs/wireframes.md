## WF-001 메인 홈

### 레이아웃 구조 (PC)
```
[GNB: 로고 | 카테고리 메뉴 | 검색창 | 찜/장바구니/마이페이지]
[Hero Banner: 시즌 프로모션 풀 배너]
┌─────────────────────────────────────────┐
│ 🤖 AI 맞춤 추천 (나를 위한 추천)          │
│ [상품카드] [상품카드] [상품카드] [상품카드] │
│ 각 카드에 "추천 이유" 뱃지 표시           │
└─────────────────────────────────────────┘
[카테고리 퀵 네비: 상의|하의|아우터|원피스|신발|가방|악세|모자]
┌──────────────────┬──────────────────────┐
│ 📈 인기 급상승    │ 🆕 신상품             │
│ 1. [이미지] 상품명 │ [2열 상품 그리드]     │
│ 2. [이미지] 상품명 │                      │
│ 3. [이미지] 상품명 │                      │
└──────────────────┴──────────────────────┘
[Footer]
```

### 핵심 컴포넌트
- `<HeroBanner />` — 자동 슬라이드, 최대 5장
- `<AIRecommendSection />` — Claude API 연동, 개인화 추천 4~8개
- `<CategoryQuickNav />` — 아이콘 + 레이블
- `<TrendingProducts />` — 실시간 인기 상품 랭킹
- `<NewArrivals />` — 최신 등록 상품 그리드

### 모바일 변경사항
- GNB: 로고 + 검색 + 아이콘만 표시
- 하단 탭바: 홈|검색|카테고리|찜|MY
- AI 추천: 가로 스크롤 카드 3개
- 카테고리: 가로 스크롤

### 레이아웃 구조 (PC)
```
[GNB]
[브레드크럼: 홈 > 상의 > 티셔츠]

[활성 필터 태그: 캐주얼 × | M사이즈 × | 2~5만원 ×] [전체 초기화]
┌────────────┬──────────────────────────────────┐
│ 필터 사이드바│ 상품 그리드 (3열)                  │
│            │ [카드][카드][카드]                  │
│ ▸ 카테고리  │ [카드][카드][카드]                  │
│ ▸ 가격대   │ [카드][카드][카드]                  │
│ ▸ 사이즈   │                                   │
│ ▸ 색상     │ [페이지네이션: 1 2 3 4 5]            │
│ ▸ 브랜드   │                                   │
└────────────┴──────────────────────────────────┘
```

### 상품 카드 구성
- 이미지 (상단)
- 찜하기 버튼 (이미지 우상단)
- 브랜드명 (작은 텍스트)
- 상품명
- 가격
- 평점 뱃지

### 모바일 변경사항
- 필터: 하단 시트(bottom sheet)로 변경
- 상품 그리드: 2열
- 무한 스크롤 적용

## WF-003 상품 상세

### 레이아웃 구조 (PC)
```
[GNB]
[브레드크럼]
┌─────────────────────┬──────────────────────────┐
│ 이미지 갤러리 (45%)  │ 상품 정보 (55%)            │
│ [메인 이미지]        │ [브랜드명 뱃지]              │
│ [썸네일 × 4]        │ [상품명 (대)]               │
│                     │ [평점★ 리뷰수]              │
│                     │ [가격]                     │
│                     │ ─────────────────          │
│                     │ 색상: ● ● ● ●              │
│                     │ 사이즈: XS S [M] L XL       │
│                     │ 수량: [ - 1 + ]            │
│                     │ ─────────────────          │
│                     │ 총 상품금액: 39,000원        │
│                     │ [♥ 찜하기] [바로 구매]       │
│                     │ [장바구니 담기]              │
└─────────────────────┴──────────────────────────┘
[탭: 상품설명 | 리뷰(243) | Q&A | 배송/반품]
[리뷰 요약: 평점 4.7 / 별점 분포 바차트]
```

### 핵심 컴포넌트
- `<ImageGallery />` — 메인 이미지 + 썸네일 스크롤
- `<OptionSelector />` — 색상/사이즈 선택, 재고 실시간 표시
- `<ReviewSection />` — 평점 분포 차트 + 리뷰 목록
- `<AISimilarProducts />` — Claude API 기반 유사 상품

## WF-004/005 장바구니 & 결제

### 스텝 표시
```
[1. 장바구니] ──── [2. 주문서 작성] ──── [3. 결제 완료]
```

### 장바구니 레이아웃
```
┌──────────────────────────────┬────────────────────┐
│ 상품 목록                     │ 주문 요약            │
│ ☑ [이미지] 상품명 / M/화이트  │ 상품 금액: 123,000   │
│          [ - 1 + ]  39,000원 │ 배송비:      3,000   │
│ ☑ [이미지] 상품명 / L/블랙   │ 쿠폰 할인:  -5,000   │
│          [ - 2 + ]  78,000원 │ ─────────────────   │
│                               │ 총 결제: 121,000원   │
│                               │ [주문하기 (2개)]      │
└──────────────────────────────┴────────────────────┘
```

### 결제 페이지 구성
- 배송지 정보 (받는 분 / 연락처 / 주소 / 상세주소 / 메모)
- 결제 수단 선택 (신용카드 | 카카오페이 | 네이버페이 | 계좌이체)
- 최종 결제 금액 확인 박스
- [결제 완료하기] CTA 버튼

## WF-010 관리자(Admin) 대시보드

### KPI 6종
- 총 회원 수 / 오늘 가입 / 활성 셀러 / 오늘 GMV / 신고 접수 / AI 추천 CTR

### 주요 패널
1. **일별 GMV 차트** — 30일 바 차트
2. **카테고리별 매출 비중** — 파이 차트
3. **셀러 승인 대기** — 승인/거절 버튼
4. **리뷰 신고 접수** — 내용 확인/삭제
5. **AI 추천 성능** — CTR / CVR / 찜 전환율 / 정확도 프로그레스 바
6. **회원 목록 테이블** — 검색/필터/정지 처리
