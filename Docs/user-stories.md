# 사용자 스토리 (User Stories)

> 버전: v1.0 | 구매자 14개 + 셀러 11개 = 총 25개

---

## 🛒 구매자 (Buyer) 스토리

### B-001 이메일 회원가입
- **역할**: 신규 방문자
- **스토리**: 이메일과 비밀번호로 회원가입하고 싶다. AI 추천 서비스를 이용하기 위해.
- **인수 조건**:
  - 이메일/비밀번호/닉네임 입력 폼 제공
  - 중복 이메일 가입 불가
  - 비밀번호 8자 이상, 영문+숫자 조합
  - 가입 완료 후 취향 설문 페이지로 이동
- **우선순위**: 높음 | **공수**: 3일

### B-002 소셜 로그인
- **역할**: 기존 SNS 계정 보유 구매자
- **스토리**: 카카오/구글 계정으로 빠르게 로그인하고 싶다.
- **인수 조건**:
  - 카카오, 구글 소셜 로그인 버튼 제공
  - 최초 로그인 시 닉네임 설정 화면 표시
  - 동일 이메일 계정 연동
- **우선순위**: 중간 | **공수**: 2일

### B-003 취향 설문으로 첫 AI 추천 받기
- **역할**: 신규 가입 구매자
- **스토리**: 스타일 취향을 입력해서 AI가 처음부터 맞는 옷을 추천받고 싶다.
- **인수 조건**:
  - 선호 스타일/사이즈/가격대 선택 (3~5단계)
  - 설문 완료 즉시 맞춤 추천 상품 노출
  - 마이페이지에서 언제든 수정 가능
  - 건너뛰기 가능 (기본 인기상품 노출)
- **우선순위**: 높음 | **공수**: 2일

### B-004 카테고리로 상품 탐색
- **역할**: 쇼핑 중인 구매자
- **스토리**: 상의/하의/아우터 등 카테고리별로 상품을 탐색하고 싶다.
- **인수 조건**:
  - 상단 내비에 대분류 카테고리 표시
  - 카테고리 클릭 시 상품 그리드 노출
  - 무한 스크롤 또는 페이지네이션 지원
- **우선순위**: 높음 | **공수**: 3일

### B-005 키워드 검색
- **역할**: 특정 상품을 찾는 구매자
- **스토리**: 키워드를 입력해서 원하는 상품을 검색하고 싶다.
- **인수 조건**:
  - 상단 고정 검색창 (모든 페이지)
  - 입력 시 자동완성 추천어 표시
  - 상품명/브랜드명/태그 기반 검색
  - 결과 없을 경우 안내 + 추천 상품 표시
  - 최근 검색어 저장
- **우선순위**: 높음 | **공수**: 3일

### B-006 필터/정렬
- **역할**: 조건이 구체적인 구매자
- **스토리**: 가격대/사이즈/색상으로 필터링하고 정렬해서 원하는 상품을 찾고 싶다.
- **인수 조건**:
  - 가격/카테고리/사이즈/색상/브랜드 다중 필터
  - 최신순/인기순/낮은가격순/높은가격순/리뷰순 정렬
  - 적용 필터 태그 표시 + 개별 삭제
  - 전체 초기화 버튼
- **우선순위**: 높음 | **공수**: 3일

### B-007 상품 상세 확인
- **역할**: 구매를 고려 중인 구매자
- **스토리**: 상품 상세 정보와 리뷰를 충분히 확인하고 싶다.
- **인수 조건**:
  - 다중 이미지 갤러리 + 상품 설명
  - 옵션 선택 시 재고 즉시 표시
  - 구매자 리뷰 목록 (평점/텍스트/이미지)
  - 사이즈 가이드 팝업
  - 셀러 스토어 링크 + 평점 표시
- **우선순위**: 높음 | **공수**: 3일

### B-008 찜하기
- **역할**: 구매 결정 전 구매자
- **스토리**: 관심 상품을 찜 목록에 저장하고 싶다.
- **인수 조건**:
  - 상품 카드/상세 페이지에 하트 아이콘
  - 클릭 시 즉시 위시리스트 추가
  - 마이페이지에서 전체 조회
  - 찜 데이터는 AI 추천 학습에 활용
  - 비로그인 시 로그인 안내 팝업
- **우선순위**: 높음 | **공수**: 2일

### B-009 장바구니/결제
- **역할**: 구매 의사가 있는 구매자
- **스토리**: 사이즈/수량 선택 후 장바구니에 담고 결제하고 싶다.
- **인수 조건**:
  - 사이즈/색상/수량 선택 후 담기
  - 수량 변경/삭제 가능
  - 총 결제금액 실시간 업데이트
  - 카드/계좌/카카오페이 결제
  - 결제 완료 시 주문확인 페이지 + 이메일
- **우선순위**: 높음 | **공수**: 4일

### B-010 주문/배송 확인
- **역할**: 주문 완료 구매자
- **스토리**: 마이페이지에서 주문 현황과 배송 상태를 확인하고 싶다.
- **인수 조건**:
  - 마이페이지에서 전체 주문 목록 조회
  - 주문 상태 단계별 표시
  - 운송장 번호로 택배 추적 연동
  - 배송 상태 변경 시 알림
  - 구매확정 버튼 (7일 후 자동)
- **우선순위**: 높음 | **공수**: 3일

### B-011 리뷰 작성
- **역할**: 구매 확정 구매자
- **스토리**: 구매한 상품에 리뷰를 작성하고 싶다.
- **인수 조건**:
  - 구매 확정 완료 건에만 리뷰 버튼 활성화
  - 별점/텍스트/이미지(최대 3장) 작성
  - 상품 상세 페이지에 리뷰 노출
  - 1회 수정 가능
- **우선순위**: 중간 | **공수**: 2일

### B-012 AI 맞춤 추천
- **역할**: 활발히 이용하는 구매자
- **스토리**: AI가 내 취향에 맞는 상품을 홈에서 추천해주면 좋겠다.
- **인수 조건**:
  - 홈 상단 "나를 위한 추천" 섹션
  - 취향설문 + 찜 + 검색 + 구매이력 종합 추천
  - 각 추천 상품에 추천 이유 문구 표시
  - 데이터 없을 경우 인기상품 대체 노출
- **우선순위**: 높음 | **공수**: 5일

### B-013 상세페이지 유사 상품 추천
- **역할**: 상품 탐색 중인 구매자
- **스토리**: 현재 보는 상품과 유사한 스타일 상품을 추천받고 싶다.
- **인수 조건**:
  - 상품 상세 하단 "비슷한 상품" 섹션
  - 카테고리/태그/가격대 기반 유사 상품
  - 4~8개 표시, 찜하기 버튼 포함
- **우선순위**: 중간 | **공수**: 3일

### B-014 취향 정보 수정
- **역할**: 취향이 변한 기존 구매자
- **스토리**: 마이페이지에서 취향 정보를 언제든 업데이트하고 싶다.
- **인수 조건**:
  - 마이페이지 > 취향 설정에서 수정 가능
  - 수정 완료 시 즉시 AI 추천 반영
- **우선순위**: 중간 | **공수**: 1일

## 🏪 셀러 (Seller) 스토리

### S-001 셀러 회원가입 및 승인
- **역할**: 입점하려는 브랜드 담당자
- **스토리**: 셀러 계정을 신청하고 관리자 승인을 받아 판매를 시작하고 싶다.
- **인수 조건**:
  - 일반 가입 후 셀러 전환 신청
  - 사업자등록번호/대표자/통장 정보 입력
  - 승인 대기 상태 마이페이지 표시
  - 승인 완료 시 이메일 알림 + 대시보드 활성화
- **우선순위**: 높음 | **공수**: 2일

### S-002 상품 등록
- **역할**: 상품을 판매하려는 셀러
- **스토리**: 판매할 의류의 정보를 입력하고 이미지를 업로드해서 상품을 등록하고 싶다.
- **인수 조건**:
  - 상품명/카테고리/가격/설명/이미지(최대 10장) 입력
  - 사이즈/색상별 재고 수량 입력
  - AI 추천용 스타일 태그 최대 5개 선택
  - 임시 저장 기능
  - 등록 후 즉시 판매중 또는 예약 등록
- **우선순위**: 높음 | **공수**: 4일

### S-003 상품 수정/상태 관리
- **역할**: 등록 상품 관리하는 셀러
- **스토리**: 가격 변경, 설명 수정, 품절 처리 등을 언제든 할 수 있어야 한다.
- **인수 조건**:
  - 전체 등록 상품 목록 조회
  - 상품명/가격/설명/이미지/태그 수정
  - 판매중/품절/숨김 상태 변경
  - 삭제 후에도 기존 주문 내역 조회 가능
- **우선순위**: 높음 | **공수**: 2일

### S-004 재고/옵션 관리
- **역할**: 다양한 옵션 운영 셀러
- **스토리**: 사이즈/색상별 재고를 개별 관리하고 싶다.
- **인수 조건**:
  - 사이즈/색상 조합별 재고 수량 입력
  - 재고 5개 이하 시 경고 알림
  - 재고 0 시 해당 옵션 자동 품절
- **우선순위**: 높음 | **공수**: 3일

### S-005 신규 주문 처리
- **역할**: 주문을 받은 셀러
- **스토리**: 신규 주문을 즉시 확인하고 발송 처리를 할 수 있어야 한다.
- **인수 조건**:
  - 신규 주문 발생 시 이메일/앱 알림
  - 신규/처리중/완료 주문 구분 표시
  - 주문 상세에서 구매자/배송지/상품 확인
  - 일별/주별 주문 수 통계 차트
- **우선순위**: 높음 | **공수**: 3일

### S-006 주문 취소/환불 처리
- **역할**: 취소 요청 받은 셀러
- **스토리**: 취소 요청을 검토하고 승인/거절할 수 있어야 한다.
- **인수 조건**:
  - 취소 요청 시 셀러 알림
  - 배송 전 취소는 자동 처리
  - 배송 후 취소는 셀러 승인/거절
  - 승인 시 자동 환불
- **우선순위**: 높음 | **공수**: 3일

### S-007 운송장 등록/배송 처리
- **역할**: 상품 발송하는 셀러
- **스토리**: 택배사와 운송장 번호를 등록해서 구매자가 배송을 추적하게 하고 싶다.
- **인수 조건**:
  - 택배사 선택 + 운송장 번호 입력
  - 등록 시 주문 상태 "배송중" 자동 변경
  - 구매자에게 배송 시작 알림
  - 다수 주문 일괄 운송장 등록
- **우선순위**: 높음 | **공수**: 2일

### S-008 셀러 스토어 페이지
- **역할**: 브랜드 이미지 관리 셀러
- **스토리**: 브랜드 소개/로고/대표 이미지를 설정한 스토어 페이지를 운영하고 싶다.
- **인수 조건**:
  - 고유 스토어 URL 발급 (/store/brandname)
  - 브랜드명/로고/배너/소개글 설정
  - 전체 등록 상품 목록 표시
  - 팔로우 기능
- **우선순위**: 중간 | **공수**: 3일

### S-009 매출/정산 확인
- **역할**: 수익 관리하는 셀러
- **스토리**: 기간별 매출 현황과 정산 금액을 확인하고 싶다.
- **인수 조건**:
  - 일별/주별/월별 매출/주문 건수 차트
  - 상품별 판매량과 매출 기여도
  - 정산 예정 금액과 수수료 명세
  - 정산 내역 엑셀 다운로드
- **우선순위**: 높음 | **공수**: 3일

### S-010 리뷰 확인/응답
- **역할**: 고객 관계 중시 셀러
- **스토리**: 고객 리뷰를 확인하고 답변을 달고 싶다.
- **인수 조건**:
  - 대시보드에 최근 리뷰 목록
  - 별점 분포/평균 평점 시각화
  - 리뷰당 1회 댓글 작성 가능
  - 부적절 리뷰 신고 기능
- **우선순위**: 중간 | **공수**: 2일

### S-011 상품별 판매 성과
- **역할**: 상품 전략 수립 셀러
- **스토리**: 각 상품의 조회수/찜/전환율을 파악하고 싶다.
- **인수 조건**:
  - 상품별 조회수/찜/장바구니/구매 수
  - 조회 대비 구매 전환율 (%)
  - 기간 필터 (7일/30일/90일)
- **우선순위**: 중간 | **공수**: 2일
